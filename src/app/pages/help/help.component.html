History
Version 0.7 - April 23, 2022
  • Added ability to connect to openMSX to reset or switch disks and tapes
  • Added ability to undo deletes and updates
  • Added ability to sort games in the table by name, company and year
  • Added Italian and Japanese translations
  • Few fixes and optimizations
Version 0.6 - March 20, 2022
  • View top played games in the dashboard
  • Added translations (Spanish, Portuguese, French in addition to English)
  • Added right-click menu for games in the list for quick access to actions
  • Few other UI changes and bug fixes
Version 0.5 - November 27, 2021
  • Add selected game to favorites list and access them from the favorites menu
  • View game launch history in the dashboard
  • Few other bug fixes
Version 0.4 - October 2, 2021
  • Drag and drop files and folders on main screen to scan
  • Manage listings by renaming, deleting or merging them
  • Many small fixes and enhancements (UI, scanning algorithm, etc.)
Version 0.3a - April 29, 2021
  • Made necessary code changes to get the Linux and Mac builds working
  • Fixed a minor bug where the last game in the list was out of screen view
Version 0.3 - April 24, 2021
  • Added option to launch games on WebMSX offline mode (and embedded within Novo Player)
  • Play music samples for some games (using an optional package of MP3 files that can be downloaded separately)
Version 0.2 - April 10, 2021
  • Improved user interface
  • Ability to rename games inline and edit its data
  • Dashboard that shows totals and running games
  • Ability to search for games by name or SHA1 code
  • Ability to move selected game to a different selection
Version 0.1 - January 13, 2021
  • Scan harddisk to find games
  • Show details and screenshots of selected game
  • Go the first matched game in the list when typing its first few letters
To do (in no particular order)
  • Filters
  • MSX News
  • Related games
  • Multiple selection and operations such as move and delete
  • blueMSX support
